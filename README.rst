FisherMartix_TD.py
===========

A code to compute Fisher matrix for time delay observable Tc.
From a project for PHY 250 Special Topic (Instructor: David Wittman, UC Davis)

project reference:
Dan Coe & Leonidas A. Moustakas (2009)
http://iopscience.iop.org/0004-637X/706/1/45


Run the code
------------

Just run

::

    python FisherMatrix_TD.py

The Fisher matrix will print out in the terminal screen.


Change the code
————————————————

The fiducial cosmology, experiment design, and mock lenses settings are hardcoded in the beginning of the code. You can change the default setting to compute new Fisher matrices.
